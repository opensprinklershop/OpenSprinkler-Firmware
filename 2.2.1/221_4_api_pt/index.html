Referência da API do Firmware 2.2.1(4)

Para acesso e controle detalhado do OpenSprinkler, consulte os endpoints da API no manual principal e no Adendo de API e Plataformas OpenSprinklerPro.

Parâmetros Gerais de Uso

  • Autenticação (pw): Todas as chamadas de API requerem a senha do dispositivo enviada por meio do parâmetro pw criptografada em um hash MD5 em caixa baixa.
  • IP do Dispositivo: Modifique o endereço IP os-ip conforme o endereço ativo configurado no seu roteador.

A documentação detalhada completa em inglês está disponível no Manual de API Principal.
