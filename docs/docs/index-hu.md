# OpenSprinkler Shop szoftver és dokumentáció

Ez az oldal az OpenSprinkler firmware-ével és az OpenSprinklerShop által karbantartott OpenSprinklerPro bővítményekkel kapcsolatos dokumentációt tartalmazza.

!!! info "OpenSprinklerShop firmware verzió"
    Az OpenSprinklerPro-t is magában foglaló OpenSprinklerShop firmware egy egyedi szoftververzió-számozást használ, amely független a hivatalos upstream OpenSprinkler szoftvercsaládtól. A jelenlegi OpenSprinklerShop firmware verzió: **2.4.0(208)**.

!!! tip "Szoftverfrissítések és portál"
    A szoftverletöltések, archívumok, változásnapkák (changelog) és a frissítési útmutatók megtalálhatók az [OpenSprinklerShop frissítési portálon](https://opensprinklershop.de/upgrade).

!!! note "Eredeti Upstream szoftver"
    A hivatalos, eredeti OpenSprinkler nyílt forráskódú firmware és a hozzá tartozó dokumentáció az alábbi címen érhető el: [https://opensprinkler.github.io/OpenSprinkler-Firmware](https://opensprinkler.github.io/OpenSprinkler-Firmware)

* Felhasználói kézikönyv: [Magyar](2.2.1/221_4_manual_hu.md), [English](2.2.1/221_4_manual.md), [Deutsch](2.2.1/221_4_manual_de.md), [Français](2.2.1/221_4_manual_fr.md), [Italiano](2.2.1/221_4_manual_it.md), [Português](2.2.1/221_4_manual_pt.md), [Polski](2.2.1/221_4_manual_pl.md).
* OpenSprinklerPro kiterjesztések: [Magyar](opensprinklerpro_hu.md), [English](opensprinklerpro.md), [Deutsch](opensprinklerpro_de.md), [Français](opensprinklerpro_fr.md), [Italiano](opensprinklerpro_it.md), [Português](opensprinklerpro_pt.md), [Polski](opensprinklerpro_pl.md).
* Szoftver verzióinformációk: [OpenSprinklerShop Firmware kiadások](https://opensprinklershop.de/upgrade/).
