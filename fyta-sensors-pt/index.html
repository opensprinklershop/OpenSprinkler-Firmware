Sensores de Plantas FYTA

Configuração rápida (Português)

Os sensores FYTA integram a umidade e a temperatura do solo a partir da nuvem FYTA no OpenSprinkler. Os valores podem ser usados para ajustes de programa por sensor, monitores e notificações.

  1. Abra https://web.fyta.de/api-token, faça login e copie o token da API.
  2. Na interface do OpenSprinkler, abra Settings -> Sensor Settings -> FYTA Setup e insira o token da API (recomendado) ou email/senha do FYTA.
  3. Adicione um novo sensor e escolha FYTA Moisture ou FYTA Temperature.
  4. Clique em Select FYTA plant and sensor, escolha uma planta com um sensor FYTA atribuído e salve.
  5. Use o valor em ajustes de programa ou monitores. Intervalo de consulta recomendado: 15–60 minutos.

Notas da plataforma: ESP32/ESP32-C5 e OSPi usam HTTPS; ESP8266 usa HTTP devido aos limites de TLS da plataforma.

Requisitos

  • Conta FYTA com pelo menos uma planta registrada e um sensor FYTA atribuído.
  • OpenSprinkler ESP32 ou OSPi (ESP8266 é suportado, mas apenas via HTTP).
  • Conexão Wi-Fi/rede ativa do controlador.

Passo 1: Informar as credenciais da FYTA

A conexão com a nuvem FYTA requer um token de API (recomendado) ou seus dados de login do FYTA (E-mail + Senha).

Token da API (recomendado)

  1. Abra https://web.fyta.de/api-token no seu navegador.
  2. Faça login com sua conta FYTA.
  3. Copie o token exibido.

Inserir no OpenSprinkler

  1. Abra a interface web do OpenSprinkler.
  2. Navegue até Definições → Configurações do Sensor.
  3. Expanda a área "FYTA Setup".
  4. Clique em "Setup FYTA credentials".

O diálogo solicitará os seguintes campos:

Campo Descrição
API Token Cole aqui o seu token gerado (recomendado)
E-Mail Alternativa: seu e-mail de login da FYTA
Password Alternativa: sua senha da FYTA

Nota: Se um token de API for inserido, o e-mail e a senha serão ignorados. Deixe e-mail e senha em branco se estiver usando um token.

  1. Clique em "Submit" – as credenciais serão salvas.

Passo 2: Adicionar o sensor FYTA

  1. Vá em Definições → Configurações do Sensor → Novo Sensor (Add Sensor).
  2. Escolha como Tipo (Type):
  3. "FYTA Moisture" – para umidade do solo (unidade: %)
  4. "FYTA Temperature" – para temperatura do solo (unidade: °C ou °F conforme a configuração da conta FYTA)
  5. Clique no botão "Select FYTA plant and sensor".

O OpenSprinkler se conectará à nuvem FYTA e carregará a sua lista de plantas.

Passo 3: Selecionar a Planta

Após o clique, aparecerá uma lista de todas as plantas da sua conta FYTA com imagem de pré-visualização, apelido e nome científico.

Somente as plantas que têm um sensor FYTA atribuído serão exibidas.

  1. Clique na planta desejada.
  2. O Nome e a ID da planta serão inseridos automaticamente no formulário.
  3. Ajuste o nome do sensor se desejar.
  4. Clique em Salvar.
